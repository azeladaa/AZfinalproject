Step 1: Browser Check

It starts when the browser checks the cache for a DNS(Domain Name System) record to find the corresponding IP address of the site.

The browser then checks four caches:

Browser Cache

The repository of DNS records for a fixed duration for websites you have previously visited.

OS Cache

if it is not found in the browser cache, the browser will make a system call.

Router Cache

Maintains its own cache of DNS records.
