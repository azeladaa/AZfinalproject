Step 5: Get Request

Once the TCP connection is established, the browser will send a GET request asking for the url web page. If you’re entering credentials or submitting a form, this could be a POST request.

It reads the request, its headers, and cookies to check what is being requested. Then it will assemble a response.

Next, the HTML framework to check the HTML tags and sends out GET requests for additional elements on the web page, such as images. CSS stylesheets, JavaScript files, etc.

These static files are cached by the browser so it doesn’t have to get them again the next time the page is visited.

Success!: Your Webpage now Appears

All these steps happen within milliseconds before we could even notice!
